#####  一个使用java Applet开发的小游戏

**1.游戏名称**： EatEat

**2.游戏操作：**

​       -> 左移 

​       <- 右移

**3.游戏说明：**

​     人物（颜团子）左右移动，接到笑脸加分，分数由笑脸的大小决定，

​    接到导弹就Game Over！

**4.程序入口：** MyApplet

**5.程序结构**： 

- src 源码文件 

  - MyApplet.java  游戏入口
  - MyPanel.java  绘画面板
  - YanTuanZi.java 颜团子类(主人物)
  - SmilingFace.java 笑脸
  - Ship.java    飞机
  - Missile.java   导弹

- res 资源文件
